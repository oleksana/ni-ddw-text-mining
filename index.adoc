= HW2 - Text Mining/NLP

== Data Description

* **Dataset**: NLTK Inaugural Addresses Corpus
* **Source**: US Presidential inaugural speeches (George Washington 1789 - present)
* **Sentences processed**: 600
* **Content**: Political speeches on governance, economy, security, and society

== How to Run

[source,bash]
----
cd src
pip install -r requirements.txt
python main.py
----

MLflow results are saved locally to `src/mlruns/`.

==== Comparison
* **Method agreement**: 68%
* NLTK uses lexicon-based approach with explicit neutral class
* HF uses transformer model with binary classification only
* NLTK tends to classify more sentences as positive/neutral
* HF forces binary decision, resulting in more negative classifications

== Issues During Implementation

1. **Token limits**: HF models limited to 512 tokens; required truncation
2. **No neutral class**: HF sentiment model lacks neutral, complicating comparison
3. **Taxonomy mismatch**: NLTK GPE vs HF LOC not directly comparable
